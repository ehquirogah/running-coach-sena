# 01. Introducción al Proyecto: Running Coach

## 📝 Descripción General
**Running Coach** es una plataforma integral de gestión y seguimiento para el entrenamiento de corredores. Diseñada para centralizar la comunicación entre atletas y entrenadores, la aplicación permite la creación de planes personalizados, el registro detallado de sesiones de entrenamiento y la visualización de métricas de rendimiento en tiempo real.

El proyecto busca profesionalizar el seguimiento del running, eliminando el uso de hojas de cálculo dispersas y proporcionando herramientas analíticas avanzadas tanto para el corredor aficionado como para el profesional.

---

## 🎯 Objetivo del Sistema
Optimizar el proceso de entrenamiento mediante la digitalización de la carga de trabajo, permitiendo:
1.  **A los Atletas:** Visualizar su progreso, registrar sus sensaciones (nivel de esfuerzo) y cumplir con sus objetivos semanales.
2.  **A los Entrenadores:** Gestionar múltiples atletas simultáneamente, asignar planes dinámicos y analizar el rendimiento histórico para ajustar cargas.
3.  **A los Administradores:** Mantener el control de acceso y la integridad de los usuarios del sistema.

---

## 🏗️ Contexto del Proyecto
Este proyecto ha sido desarrollado bajo un enfoque **Full-Stack** moderno utilizando **Supabase (PostgreSQL)** como núcleo de persistencia, priorizando la velocidad de carga (Performance), la estética visual (Premium UI) y la robustez de los datos mediante el cumplimiento estricto de reglas de negocio predefinidas.

---

## 🔑 Valores Agregados
- **Interfaz Premium:** Uso de glassmorphism, gradientes suaves y micro-animaciones para una experiencia de usuario superior.
- **Modo Demo:** Capacidad de exploración total sin necesidad de una base de datos activa mediante un sistema de mocks avanzado.
- **Reporting Avanzado:** Generación automática de resúmenes de distancia, tiempo y esfuerzo.
- **Mobile Friendly:** Diseño responsivo que permite al atleta registrar su sesión inmediatamente después de correr desde su celular.
